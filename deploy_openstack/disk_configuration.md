# 配置节点磁盘

发现并增加所有节点后，需要分配各节点的磁盘空间。

## Controller节点

* 勾选全部**Controller节点**，点击**磁盘配置**按钮。

  ![openstack_install_451](../images/openstack_install_451.png)

* 配置磁盘分区，如下图所示，**无特殊需求建议使用默认分区方案**。

  ![openstack_install_462](../images/openstack_install_462.png)
